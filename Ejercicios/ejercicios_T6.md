###Ejercicio 3. Buscar información acerca de los tipos de ataques más comunes en servidores web, en qué consisten, y cómo se pueden evitar

- **Ataque Por Injection:** es una técnica para modificar una cadena de consulta de base de datos mediante la inyección de código en la consulta.

- **DDoS:** ataques de denegación de Servicio mediante la saturación del servidor con el envío constante de peticiones.

- **Fuerza Bruta:** intento de romper usario y contraseña probando combinaciones enormes de las mismas.

- **Cross Site Scripting:** Los atacantes utilizan Cross-site Scripting (XSS) para inyectar scripts maliciosos en lo que serían sitios web inofensivos. 

Para evitarlos es recomendable tener un DMZ bien configurado, donde además no dejemos abierto ningún puerto innecesario. Tips de seguridad básicos o complejos pero nada específico para cada ataque, un servidor
debe ser seguro en todas las situaciones.
